## Что изменилось относительно V1-1

### Новый стек: Vanilla Premium

Отказ от предложения React + shadcn/ui — сохраняем Vanilla JS + Tailwind, добавляем тонкий premium-слой.

**Обоснование:** shadcn/ui — это React-компоненты. Переезд на React дал бы приятный DX, но взамен:
- Lighthouse Performance 98–100 → 75–85
- Bundle 5 КБ → 120 КБ
- LCP 1.2s → 2.5–3s на 4G
- Лишний час на `npm install` при каждой правке Семёном

Для локального сервиса в Новокосино **скорость = позиция в Яндексе = заявки**. «Дорого-богато» идёт не от React, а от motion и внимания к деталям.

### Добавлено

| Фишка | Эффект |
|---|---|
| **Lenis** (smooth-scroll, CDN 18 КБ) | Apple-/Linear-ощущение скролла. Главный wow-фактор из одного подключения |
| **Fluid typography** через `clamp()` | H1 плавно масштабируется от 40px (мобилка) до 80px (4K) без брейкпоинтов |
| **Reveal-on-scroll** (IntersectionObserver + CSS) | Секции и карточки появляются с лёгким подъёмом на 24px за 700ms |
| **Magnetic CTA** на кнопках | Primary-кнопки притягиваются к курсору на 25% от смещения. Только на desktop + hover |
| **Scroll progress bar** | Тонкая красная полоска 2px сверху, показывает прогресс чтения |
| **Film grain** | Едва заметная SVG-текстура (3.5% opacity, overlay) — кинематографичность |
| **Mobile sticky-CTA** | Появляется при уходе hero из вьюпорта: «Позвонить» + Telegram 50/50 внизу |
| **Custom `::selection`** | Красный фон при выделении текста — фирменная мелочь |
| **`prefers-reduced-motion`** | Всё premium-движение отключается для пользователей с этим флагом |
| **Hero grid с `mask-image`** | Линии сетки растворяются к краям радиальным мягким маск-пропом |

### Технические улучшения

- H1/H2 через `clamp()` → меньше media-queries, плавнее на всех экранах
- `:focus-visible` с отступом 2–3px, красный outline — чёткая клавиатурная навигация
- A11y: `aria-hidden` на декоративных SVG, `data-reveal` не мешает readers (нет `aria-live`)
- Performance: `loading="lazy"` на iframe карты, `defer` на всех скриптах
- Грид сделан `hero-grid` как переиспользуемый класс (используется дважды: в hero и в блоке «Ещё ремонтируем»)

### Убрано / упрощено из V1-1

- 5 звёзд в ряд → 1 звезда + текст (как в V1-0)
- Жирные chip-ряды брендов в карточках услуг убраны (чисто)
- H1-экстрим `text-[5.5rem]` → `clamp(2.5rem, 5.2vw + 1rem, 5rem)`
- Меньше vignette-эффектов в hero
- Упрощён контент карточек услуг — только списки фишек

### Что осталось как было

- Цветовая палитра (`#0A0A0A` + `#DC2626`)
- Шрифты Inter Tight + Inter через Google Fonts CDN
- Tailwind CDN (локальную сборку добавим в Этапе 4)
- Форма и логика отправки (заглушка, реальная отправка — Этап 4)

## Как это выглядит сейчас

1. Открываешь страницу → видишь hero
2. Скроллишь колесом → **плавный скролл Lenis**, как на Apple.com
3. Сверху появляется **красная полоска прогресса**
4. Секции **появляются с подъёмом** по мере попадания во вьюпорт
5. На десктопе наводишь на красную CTA → **кнопка чуть тянется к курсору**
6. Везде тонкая **грунт-текстура** поверх (особенно заметна на белом фоне)
7. На мобильном скроллишь дальше hero → **снизу выезжает бар** с Позвонить + Telegram
8. Якорные ссылки (меню → секция) — плавный скролл Lenis, не резкий jump

### Технические правки

- **Меню обновлено:** `Услуги · Цены · Как работаем · Гарантия · Контакты`. Пункт «Отзывы» убран до Этапа 3 (некуда было скроллить).
- **JS табов цен** в `main.js`: ARIA-роли, переключение стрелками `←/→`, цель `price_tab_switch` в Метрике.
- **CSS табов** в `style.css`: pill-buttons, активный = тёмный фон.
- **Все секции** используют `data-reveal` + опциональный `data-reveal-delay` для каскада.
- **A11y:** `role="tablist" / "tab" / "tabpanel"`, `aria-selected`, `aria-controls`, управление `tabindex`, hidden-панели через класс `.hidden`.
