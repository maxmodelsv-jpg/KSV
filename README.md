# Сервис KSV — лендинг

Одностраничный сайт сервисного центра по ремонту телефонов, планшетов, ноутбуков и оргтехники в Москве у метро Новокосино.

## Стек

- HTML5 · Tailwind CSS (CDN на старте, сборка CLI в продакшене)
- Vanilla JavaScript (без зависимостей)
- Netlify (хостинг + функции)
- Telegram Bot API (приём заявок)
- Яндекс.Метрика · Яндекс.Карты

## Структура

```
index.html              — главная страница (one-page)
assets/
  css/style.css         — кастомные стили
  js/main.js            — бургер, cookie, трекинг
  js/form.js            — маска, валидация, отправка
research/               — бенчмарк 55 сайтов (отчёт + сырые данные)
V1-1/                   — снапшот первой версии
CLAUDE.md               — полный бриф проекта (20 ответов + этапы + чек-лист)
V1-1.md                 — консолидированная точка входа для возобновления работы
```

## Локальный запуск

```bash
npx serve .
# или открыть index.html напрямую в браузере
```

## Деплой

Push в `main` → Netlify собирает и выкладывает автоматически.

## Статус

Этап 1 из 4 (скелет) — реализован. Этапы 2–4 описаны в [CLAUDE.md](CLAUDE.md).
