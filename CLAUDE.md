# Сайт-одностраничник сервисного центра KSV

## Управление контекстом
- Если использование контекста превышает **65%** — выполнить `/compact` перед следующим крупным шагом.

## О проекте
Лендинг для сервисного центра **KSV** по ремонту телефонов, планшетов, ноутбуков и офисной техники в Москве у метро Новокосино.
Главная цель — приём заявок: посетитель звонит или отправляет заявку → звонок владельцу или Telegram-бот.

Сайт одностраничный (one-page), все секции прокручиваются якорями из меню.

## Стек
- **HTML5** — семантическая разметка
- **Tailwind CSS** — стили (через CDN на старте, потом сборка CLI)
- **Vanilla JS** — интерактив (без jQuery, без фреймворков)
- **Netlify** — хостинг + CI/CD (push в main = авто-деплой)
- **Telegram Bot API** — приём заявок из формы (второй канал)
- **Яндекс.Метрика** — аналитика + цели
- **Яндекс.Карты API** — филиал на карте в блоке «Контакты»

## Команды
- **Локальный запуск:** `npx serve .` (или Live Server в VSCode)
- **Сборка Tailwind в watch:** `npx tailwindcss -i assets/css/input.css -o assets/css/style.css --watch`
- **Сборка Tailwind в prod:** `npx tailwindcss -i assets/css/input.css -o assets/css/style.css --minify`
- **Деплой:** `git push origin main` — Netlify соберёт сам

## Секции лендинга (в порядке сверху вниз)
1. **Hero** — H1 «Ремонт техники у метро Новокосино. Экспресс от 30 минут», CTA «Позвонить» + «Telegram», бейдж 5.0 Яндекс
2. **Услуги** — 3 карточки в фокусе: Телефоны / Планшеты / Ноутбуки + блок «Ещё ремонтируем» (принтеры, МФУ, мониторы, видеокарты, картриджи)
3. **Цены** — 4 таблицы с табами (Телефоны / Планшеты / Ноутбуки / Оргтехника), колонки: Работа / Цена «от» / Срок / Гарантия
4. **Как мы работаем** — 4 шага с временными ориентирами
5. **Гарантия** — усиленный блок «Гарантируем, потому что уверены», 3 колонки по срокам
6. **Нам доверяют** — 2 карточки с рейтингами Яндекс (5.0, 319) и 2ГИС (4.6, 16)
7. **Отзывы** — карусель из 6 ручных карточек + кнопки «Все отзывы» / «Оставить отзыв»
8. **Наши работы** — галерея «до/после», placeholder до появления фото
9. **Работаем с техникой** — бегущая строка логотипов ~20 брендов, монохром → цвет на hover
10. **Контакты** — адрес, метро, часы, карта Яндекс, телефон, все мессенджеры, полная форма
11. **Футер** — реквизиты, политика (модалка), ссылки на Яндекс/2ГИС/VK

## Бриф проекта (ответы на 20 вопросов)

### География и контакты
- Один город — Москва, один адрес, **без выезда мастера**
- **Адрес:** Новокосинская ул., 19, 1 этаж (совместный вход с ломбардом), 111673
- **Метро:** Новокосино
- **Телефон:** +7 (964) 637-55-95 (главный канал связи)
- **Часы:** открытие 09:00, ориентир 09:00–21:00 ⚠ **нужно подтвердить точные дни**
- **Каналы связи:** звонок (главный) → Telegram (второй) → WhatsApp/VK (есть, без акцента)

### Услуги
- **Top-3 в фокусе:** Телефоны (iPhone + Android) / Планшеты / Ноутбуки и ПК
- **Вторичные:** Принтеры и МФУ, мониторы, видеокарты, заправка картриджей
- **Не продвигаем:** кофемашины (размывают позиционирование)

### Цены
- **Формат:** «от …» + диапазоны (вариант B)
- **4 таблицы** по категориям с табами
- **6–10 строк в каждой** — топ-работы, не весь каталог
- **Над таблицей предупреждение:** «Итоговая цена определяется после бесплатной диагностики»

### Гарантия (ключевое УТП)
- **До 12 месяцев** — на пайку, материнские платы, сложный ремонт
- **6 месяцев** — на замену дисплеев, АКБ, корпусных деталей
- **30 дней** — на программные работы
- **Оригинал / качественный аналог** — на выбор клиента
- **Покрывает:** повторный ремонт той же поломки бесплатно, иначе возврат денег

### Сроки ремонта
- **Экспресс 30–60 мин при клиенте** — замена дисплеев топовых моделей телефонов
- **20–30 мин** — замена АКБ
- **1–3 дня** — стандартные работы, ноутбуки
- **2–5 дней** — принтеры / МФУ
- **До 14 дней** — пайка, материнские платы, залитие
- **Диагностика бесплатная**, 15–30 мин при клиенте

### Форма заявки (средний набор полей)
1. Имя — текст, required
2. Телефон — маска `+7 (___) ___-__-__`, required
3. Устройство — select (Телефон / Планшет / Ноутбук / Принтер или МФУ / Другое), required
4. Проблема — textarea, optional
5. Чекбокс согласия на обработку ПДн по 152-ФЗ — required
6. Скрытый honeypot (`website`) против ботов
7. Скрытые `utm_*` параметры

**Две формы на странице:**
- Компактная в hero (имя + телефон + кнопка)
- Полная в блоке «Контакты»
- Обе → Telegram-бот с пометкой `source: "hero"` / `"contacts"`

## Договорённости по коду
- **Язык:** комментарии и коммиты — на русском; код (классы CSS, имена переменных, функций) — на английском
- **Адаптив:** mobile-first, брейкпоинты Tailwind (`sm` 640, `md` 768, `lg` 1024, `xl` 1280)
- **Целевые устройства:** iPhone SE 375×667, iPhone 14 390×844, Android 360×800, iPad 768×1024, Desktop 1440×900
- **Изображения:** только `.webp`, фото до 200 KB, иконки SVG inline
- **JS:** только Vanilla, без зависимостей. Никакого jQuery, Bootstrap JS, Lodash
- **Доступность:** `alt` у всех картинок, `aria-label` у иконочных кнопок, контрастность AA, видимый focus
- **SEO обязателен:**
  - `<title>`, `<meta name="description">`, Open Graph, Twitter Card
  - `schema.org` `LocalBusiness` + `AggregateRating`
  - `sitemap.xml`, `robots.txt`
  - Один `<h1>` на странице, логичная иерархия `<h2>`/`<h3>`
- **Производительность:**
  - Lighthouse ≥ 90 по всем метрикам на мобильном
  - Шрифты — `font-display: swap`, preload критичных
  - JS — `defer`, CSS — критичный inline

## Чего НЕ делать
