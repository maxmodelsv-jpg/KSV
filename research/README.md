# Research

Папка для бенчмарк-исследования сервисных центров СНГ.

## Структура
- `screenshots/` — скриншоты главных страниц
  - `<domain>-desktop-1440.png` — десктоп 1440×900
  - `<domain>-mobile-390.png` — мобильный 390×844 (iPhone 14)
- `report.md` — итоговый отчёт (будет создан после прогона)
- `sites.json` — сырые данные по каждому сайту (список доменов, метрики, цитаты)

## Как запустить
1. Убедиться, что Node.js установлен и `playwright` MCP подключён (`claude mcp list`).
2. Передать промпт исследования с параметрами `screenshots = true`, `target_count = 30`.
3. Результат складывается сюда.

## Правила именования
Домен без `www.` и без протокола: `example.ru` → `example.ru-desktop-1440.png`.
