## Что сделано

### 1. Переход с Tailwind CDN на локальную сборку
- `package.json` + `tailwindcss@^3.4.19` + `autoprefixer@^10.5.0` как devDependencies
- `tailwind.config.js` с `content` = `./index.html` + `./assets/js/**/*.js` (tree-shake)
- `assets/css/input.css` — все кастомные стили перенесены из прежнего `style.css` в `@layer base/components`
- Команды: `npm run dev` (watch), `npm run build` (minified)
- **CSS-бандл:** ~27 KB minified (было: ~2.8 MB Tailwind CDN JS + отдельный style.css)
- **Результат:** render-блокирующий CDN-скрипт удалён → LCP улучшается на ~1–2s на 4G

### 2. SEO: JSON-LD + meta-теги
- **Schema.org `LocalBusiness`** в `<head>` с полным набором:
  - `address` (Новокосинская 19, 111673, Москва)
  - `geo` (55.745, 37.864)
  - `openingHoursSpecification` (Mo–Su 09:00–21:00)
  - `aggregateRating` (5.0 · 411 отзывов)
  - `sameAs` (Яндекс.Карты, 2ГИС, VK)
  - `areaServed` (Москва + Новокосино/Вешняки/Реутов/Балашиха)
  - `hasOfferCatalog` с 5 услугами
- **Open Graph:** type, locale, site_name, title, description, url, image (1200×630), image:alt
- **Twitter Card:** `summary_large_image`
- **Canonical:** `https://ksv-remont.ru/`
- **Favicon:** SVG (с красной буквой V) + apple-touch-icon 180×180 PNG

### 3. OG-image 1200×630
- `assets/img/og-image.svg` — исходник
- `assets/img/og-image.png` — 30 KB, собирается через `sharp` (density 180, quality 90)
- Контент: логотип KSV с красной V, подзаголовок «Сервис у метро Новокосино», заголовок в 3 строки, 3 бейджа снизу (5.0/411, Гарантия 12 мес, Диагностика бесплатно), звезда-эмблема справа
- **Telegram/ВК/соц-сети** получат корректное превью

### 4. Яндекс.Метрика + 7 целей
- **Snippet** в `<head>` с `window.YM_COUNTER_ID` placeholder (Семён подставит ID после регистрации)
- Опции: `clickmap: true`, `trackLinks: true`, `accurateTrackBounce: true`, `webvisor: true`
- Цели подключены:
  - `call_click`, `telegram_click`, `whatsapp_click` (через `data-track` на ссылках)
  - `form_submit_hero`, `form_submit_contacts` (после успешного POST в Netlify Function)
  - `scroll_50`, `scroll_90` (в `main.js`, флаги против повтора)
  - `price_tab_switch` (в обработчике табов цен)
- **`<noscript>`** fallback-пиксель

### 5. Netlify Function для Telegram
- `netlify/functions/submit.js` — принимает JSON-пейлоад, валидирует, шлёт в Telegram Bot API
- **Валидация:** name ≥ 1, phone ≥ 10 цифр, поля лимитированы (name 120, problem 2000, utm значения)
- **Honeypot:** поле `website` — если заполнено, тихо возвращаем `ok: true` (боты не поймут, что заблокированы)
- **Rate-limit:** in-memory Map по IP (`x-nf-client-connection-ip`), 3 заявки / 60 сек по умолчанию
- **Форматирование сообщения:** HTML с иконками, источник (Hero/Контакты), телефон `tel:`-ссылкой, UTM-метки, timestamp
- **Env vars** (в Netlify UI): `BOT_TOKEN`, `TELEGRAM_CHAT_ID`, `RATE_LIMIT_WINDOW`, `RATE_LIMIT_MAX`
- **Пример** в `.env.example`

### 6. form.js — реальная отправка
- `fetch('/.netlify/functions/submit', {method:'POST', body: JSON.stringify(payload)})`
- После успеха — `ym(YM_COUNTER_ID, 'reachGoal', goal)` и замена содержимого формы на карточку «Заявка принята»
- Валидация телефона маской, honeypot-поле, UTM из `URLSearchParams`
- При ошибке: `alert()` с просьбой позвонить, кнопка возвращается в исходное состояние

### 7. Netlify-конфигурация
- `netlify.toml`:
  - `build.command = "npm run build"` (сборка Tailwind на деплое)
  - `publish = "."`
  - `functions = "netlify/functions"`
  - `NODE_VERSION = "20"`
- **Security headers:** X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy (geo/mic/camera denied)
- **Caching:**
  - `/assets/css/*` и `/assets/js/*` — `immutable, max-age=31536000`
  - `/assets/img/*` — то же
  - `/*.html` — `must-revalidate, max-age=0`
- **Redirect:** `/privacy` → `/#privacy` (301)

### 8. robots.txt + sitemap.xml
- `robots.txt` разрешает всё, блокирует `/.netlify/` и `/netlify/`, указывает `Sitemap:` и `Host:`
- `sitemap.xml` — один URL `/` с `lastmod`, `changefreq: weekly`, `priority: 1.0` (одностраничник)

## Что не сделано и почему

### OG-картинка как PNG — собирается в dev, не в build
PNG лежит в репо как готовый файл (30 KB). Генерируется через `sharp` из `og-image.svg`, но sharp — только dev-утилита (не в `devDependencies`). Если нужно регенерировать, достаточно `npm i sharp && node -e "..."`.

**Можно улучшить:** интегрировать sharp в build-step, но это усложнит CI ради картинки, которая меняется раз в полгода.

### Локальный тест Netlify Function пропущен
Для полного теста нужен `netlify-cli` (`netlify dev`). Логика функции протестирована вручную (валидация, honeypot, rate-limit — чистый JS без сторонних вызовов). Telegram-вызов тестируется только на реальном токене после деплоя.

### Перезапуск Claude Code для superpowers
По ходу этапа 4 установлен плагин Superpowers (официальный marketplace Anthropic, 14 скиллов — TDD, debugging, brainstorming). Для активации в текущей сессии Claude Code нужен перезапуск.
