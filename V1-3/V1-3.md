### 2. Секция `#works` — галерея до/после
- **6 плиток** с разными видами работ (дисплей, залитие, матрица, АКБ, термопаста, материнка)
- **Интерактивный слайдер** до/после на `<input type="range">` — Vanilla JS, без библиотек
- Плейсхолдер из двух SVG-слоёв с диагональной штриховкой + бейджи «ДО» / «ПОСЛЕ»
- Белый разделитель 2px с круглой «ручкой» и стрелками
- Range невидим поверх изображения (cursor: ew-resize), `aria-label` читаем ридером
- **Слоты под реальные фото** `.work-before` / `.work-after` — заменить `background` на `background-image: url(...)` когда появятся реальные before/after
- Под галереей подпись: «Примеры работ будут обновляться реальными фото мастерской»

### 3. Секция `#brands` — бегущая строка
- **21 бренд**: Apple, Samsung, Xiaomi, Huawei, Honor, Realme, OPPO, Sony, Google Pixel, Lenovo, ASUS, Acer, HP, Dell, MSI, MacBook, Microsoft, Canon, Epson, Brother, Kyocera
- **Бесшовная анимация** `@keyframes brands-scroll` (45s linear infinite) с дублированием списка
- **Pause on hover** (`animation-play-state: paused`)
- **Mask-image** fade-out 8% по краям (элегантный Linear-стиль)
- Тёмный фон `bg-dark` + hero-grid 30% + белые wordmark'ы на 35% opacity → 100% на hover
- **`prefers-reduced-motion`** отключает анимацию

### 4. Политика конфиденциальности — модалка `<dialog>`
- Семантический `<dialog id="privacy-modal">` с `showModal()` + backdrop blur
- Открывается по любой ссылке `href="#privacy"` (форма, футер, cookie-баннер)
- Также по прямому URL-хэшу `#privacy` (обработчик `hashchange`)
- **Закрывается:** ✕ кнопка в углу, кнопка «Понятно» внизу, клик по backdrop, Esc (нативный)
- **9 разделов** по 152-ФЗ: общие положения, какие данные собираем, цели, правовые основания, передача третьим лицам, сроки и защита, права субъекта ПДн, cookies, изменения политики
- Упоминание **Яндекс.Метрики и Вебвизора** в разделе «Цели обработки»
- Placeholder-ы `[[ORG_TYPE]]`, `[[ORG_NAME]]`, `[[INN]]` — заполнит Семён после регистрации юр. лица
- E-mail для запросов: `privacy@ksv-remont.ru`
- Редакция с датой, `max-height: 86vh`, прокрутка внутри диалога

## Технические правки

### HTML
- Между `#trust` и `#contacts` добавлены 3 новые секции
- Пункт **«Отзывы»** возвращён в desktop (`<nav>`) и mobile (`#mobile-menu`)
- Нав-меню десктопа: `gap-7 lg:gap-9` вместо `gap-9` — чтобы все 6 пунктов влезли на 768–1024
- Цифра отзывов **319 → 411** в hero и в блоке `#trust` (обновлено из Яндекс.Карт)
- **URL карточки Яндекс.Карт** — с заглушки `/org/ksv/` на реальный `/org/ksv/139027050840/`

### CSS (`style.css`)
- `.reviews-card` + кастомный scrollbar для `.reviews-track`
- Полный модуль `.work-view` / `.work-layer` / `.work-divider` / `.work-handle` / `.work-range` — плейсхолдеры, CSS clip-path, невидимый range-input
- `.brands-marquee` + `.brands-track` + `.brand-item` + `@keyframes brands-scroll`
- `.privacy-dialog` + `.privacy-dialog-inner` / `-head` / `-body` / `-foot` / `-close` + `@keyframes privacy-pop`
- `prefers-reduced-motion` — отключает `brands-scroll`

### JS (`main.js`)
- **Reviews prev/next**: `scrollBy()` на ширину первой карточки + gap
- **Works ranges**: `input` event → `--pos` custom property на `.work-view`
- **Privacy modal**: `showModal()` по клику на `a[href="#privacy"]`, закрытие по `[data-privacy-close]`, клик по backdrop, hashchange-listener для прямого URL
